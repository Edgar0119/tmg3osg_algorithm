# [37. Sudoku Solver](https://leetcode.com/problems/sudoku-solver)

## Description

<p>Write a program to solve a Sudoku puzzle by filling the empty cells.</p>

<p>A sudoku solution must satisfy <strong>all of the following rules</strong>:</p>

<ol>
	<li>Each of the digits <code>1-9</code> must occur exactly once in each row.</li>
	<li>Each of the digits <code>1-9</code> must occur exactly once in each column.</li>
	<li>Each of the digits <code>1-9</code> must occur exactly once in each of the 9 <code>3x3</code> sub-boxes of the grid.</li>
</ol>

<p>The <code>&#39;.&#39;</code> character indicates empty cells.</p>

### **Python3**

```python
class Solution:
    def solveSudoku(self, board: List[List[str]]) -> None:
        """
        Do not return anything, modify board in-place instead.
        """
        # 0 <= i, j <= 8
        def isVaild(num, i, j):
            for index in range(9):
                if board[i][index] == str(num) or board[index][j] == str(num):
                    return False

            block_i, block_j = i//3, j//3
            for index_i in range(3):
                for index_j in range(3):
                    if board[block_i*3 + index_i][block_j*3 + index_j] == str(num):
                        return False
            return True

        def dfs(i, j):
            if i == 9: #(9, 0)
                return True
            if j == 9:
                return dfs(i+1, 0)
            if board[i][j] != ".":
                return dfs(i, j+1)
            
            for num in range(1, 10):
                if isVaild(num, i, j):
                    #print(i, j, num)
                    board[i][j] = str(num)
                    if dfs(i, j+1):
                        return True
            board[i][j] = "."
        dfs(0, 0)
```
